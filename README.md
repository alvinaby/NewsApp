## NewsApp
This is a simple app to view list of news using:
* MVP Architecture
* Repository Pattern
* Retrofit
* RxJava
* Glide
* Room Persistence
* Dagger2

#### NewsApp functions:
* View list of news
* When user click a news, it will open the source (URL) of the news using Android built-in browser
* Dark Mode
